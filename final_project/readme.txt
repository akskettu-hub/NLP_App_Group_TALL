Install dependencies: pip install -r requirements.txt
Run the app: python run.py
Go to http://127.0.0.1:5000/ in your browser.
Search by typing a query.

(I haven't set any enviroment variables, because Flask can automatically detect the app when you run it through the run.py file and it's easier this way. Once we want to get ready for production we can still set them if we want)
